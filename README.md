# 🤖 OpenClaw AI Satış Asistanı Pro | v2.5

OpenClaw AI Satış Asistanı, satış ekiplerinin pazar araştırması ve aday müşteri (prospekt) bulma süreçlerini %90 oranında otomatize eden, Python tabanlı profesyonel bir yapay zeka aracıdır.

---

## 🔥 Temel Özellikler

- **⚡ Smart Start (Hibrit Keşif)**: Google Web Search ve LinkedIn ağını aynı anda tarayarak en güncel şirket listelerini saniyeler içinde oluşturur.
- **🧐 AI Etik Puanlama (0-10)**: Yapay zeka, bulunan her şirketin web sitesine girer, içeriği okur ve senin ürününle ne kadar uyumlu olduğunu puanlar.
- **📄 Akıllı Şirket Özeti**: Her aday firmanın ne iş yaptığını, hangi sektörlere odaklandığını tek cümlede raporlar.
- **✉️ Hiper-Kişiselleştirilmiş Satış Mesajları**: AI, her firma için o firmaya özel, "kanca" cümleler (Hook) barındıran satış teklifi taslakları hazırlar.
- **🛡️ Güvenli & Hızlı**: LinkedIn session verileri şifreli olarak (.env) tutulur; karmaşık teknik ayarlar "Geliştirici Modu" altında gizlenmiştir.

## 🚀 Hızlı Başlangıç

### 1. Gereksinimler
- Python 3.10+
- Docker (Gateway için)
- Ollama (LLM sunucusu)

### 2. Kurulum
Bağımlılıkları tek bir komutla yükleyin:
```bash
pip install streamlit requests
```

### 3. Çalıştırma
Sadece aşağıdaki kısayolu çalıştırarak paneli açabilirsiniz:
```bash
AnaliziBaslat.bat
```

## 🏗️ Mimari Yapı

- **UI**: Streamlit (Modern & Responsive)
- **Engine**: Parallel Discovery (Node.js Skills + Python Bridge)
- **AI Model**: Qwen 2.5 / Llama 3 (Lokal & Gizli)
